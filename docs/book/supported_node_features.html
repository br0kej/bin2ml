Supported Node Features

The bin2ml graph --feature-type flag supports a number of different node features where features are extracted from the basic blocks and are typically simple counts.

feature-typeDescriptionPaperFeaturesArchitecture
geminiGenerates 7 out of 8 of the basic blocks features from the seminal Gemini Paper.Paper (arxiv)
  • Number of Call instructions
  • Number of Transfer Instructions
  • Number of Arithmetic Instructions
  • Number of Instructions
  • Numeric Consts
  • String Consts
  • Number of Offspring
The final feature betweeness is calculated downstream natively in networkx		x86/64 / MIPS32/64 / ARM32/64
dgisGenerates the features used within DGIS paper.Paper (ieee)
  • Number of Stack operations
  • Number of arithmetic operations
  • Number of Logic Operations
  • Number of Compare operations
  • Number of Library Calls
  • Number of Unconditional Jumps
  • Number of Conditional Jumps
  • Number of Generic Instructions
x86/64 / MIPS32/64 / ARM32/64
discovreGenerates all 6 features used within the DisovRE paperPaper (ndss)
  • Number of Call instructions
  • Number of Transfer Instructions
  • Number of Arithmetic Instructions
  • Number of Instructions
  • Numeric Consts
  • String Consts
x86/64 / MIPS32/64 / ARM32/64
